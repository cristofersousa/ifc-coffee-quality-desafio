## Bem-vindo(a), Squad Colmeia!

A Specialty Coffee Association (SCA) realiza campeonatos de avaliação sensorial de cafés especiais.

Atualmente as avaliações são realizadas manualmente, dificultando a consolidação dos resultados e a visualização do ranking dos participantes.

O objetivo deste projeto é desenvolver uma aplicação web para cadastro, consulta e classificação de cafés avaliados durante uma competição inspirada na metodologia SCA.

Você faz parte da **Squad Colmeia**, equipe responsável pelo desenvolvimento Front-end do sistema **Coffee Quality Challenge 2026**.

---

## 🚀 O Desafio

O Product Owner reuniu os requisitos do negócio, os wireframes e o backlog inicial do produto.

Sua missão é desenvolver a primeira versão da aplicação utilizando Vue.js.

O prazo é curto e o cliente deseja uma solução funcional para apresentar durante o campeonato.

Você aceita o desafio?

# 🏆 Fluxo de Desenvolvimento Recomendado

Para facilitar a implementação, siga a seguinte ordem:

1. Criar o projeto Vue + Vite
2. Configurar Vue Router
3. Criar Layout Principal
4. Criar Componentes
5. Desenvolver Home
6. Desenvolver Avaliações
7. Desenvolver Ranking
8. Desenvolver Detalhes do Café
9. Desenvolver Página 404
10. Realizar Deploy
